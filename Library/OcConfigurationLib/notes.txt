    key (gives name)
    type

dict (map):
    key (does not give name)
    type
    (skip)

array:
    basic-type
    (skip)

type_or_array:
    array|type
    
type:
    dict|basic_type
    
basic-type:
    data|integer|string|true|false

===

data or integer

 - if type == "blob" nothing else needed (and size cannot be present)
 - if we have type and size, nothing else needed
 - if integer:
   - if no type, type = UINT32
   - size never set automatically
 - if data and there is data:
   - get data bytes
   - if no type, then size->type is
     - 2->UINT16
     - 4->UINT32
     - 8->UINT64
     - else: UINT8
   - if no size, and type (by now) is UINT8, then size = data bytes
